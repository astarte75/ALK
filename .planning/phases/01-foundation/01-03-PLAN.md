   ```json
           { "key": "X-XSS-Protection", "value": "1; mode=block" },
           { "key": "Referrer-Policy", "value": "strict-origin-when-cross-origin" }
         ]
       }
     ]
   }
   ```

2. Initialize git repo (if not already) and commit all files:
   ```bash
   git init
   git add -A
   git commit -m "feat(foundation): scaffold Next.js 15 with styled-components SSR, dark theme, i18n"
   ```

3. Deploy to Vercel:
   - Try `npx vercel --yes` for initial deployment (links project)
   - If Vercel CLI not installed: `npm install -g vercel` first
   - If auth required: create a checkpoint for the user to authenticate
   - After linking, run `npx vercel --prod` to deploy to production
   - Record the production URL

4. Install Playwright:
   ```bash
   npm install -D @playwright/test
   npx playwright install chromium
   ```

5. Create `playwright.config.ts`:
   - `baseURL`: use `process.env.BASE_URL || 'http://localhost:3000'`
   - `testDir`: `'./tests'`
   - `webServer`: start `npm run dev` on port 3000 when no BASE_URL is set
   - Only chromium project (speed)
   - `timeout`: 30000

6. Create `tests/smoke.spec.ts` (covers FOUND-01, FOUND-02):
   - Test: page loads and returns 200
   - Test: page has no console errors containing "hydration" or "className did not match"
   - Test: page source contains styled-components class (proves SSR injection)

7. Create `tests/theme.spec.ts` (covers FOUND-03):
   - Test: body has background-color matching #1A1E22 (rgb(26, 30, 34))
   - Test: :root has CSS custom property --color-bg
   - Test: text color matches #F9FAFB
   - Test: --color-accent-teal and --color-accent-gold are defined

8. Create `tests/i18n.spec.ts` (covers FOUND-05):
   - Test: GET / contains Italian text ("Investiamo" or content from it.json)
   - Test: GET /en contains English text ("We invest" or content from en.json)
   - Test: both pages return 200

9. Create `tests/responsive.spec.ts` (covers FOUND-06):
   - Test: page renders at viewport 320x568 without horizontal scrollbar

   - Test: content is visible at both viewports
