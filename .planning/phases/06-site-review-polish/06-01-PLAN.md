**ScrollReveal.tsx** — Create a reusable client component wrapper that reveals children on scroll:
- `'use client'` directive
- Props: `children`, `delay?: number` (default 0), `direction?: 'up' | 'left' | 'right'` (default 'up'), `className?: string`, `as?: keyof JSX.IntrinsicElements` (default 'div')
- Uses `useGSAP()` hook with `scope` on container ref
- Inside `gsap.matchMedia()`:
  - `reduceMotion` condition: `gsap.set` to final state (opacity 1, y/x 0)
  - Normal condition: `gsap.from` on children with `scrollTrigger: { trigger, start: 'top 85%', once: true }`, `opacity: 0`, `y: 40` (or x for left/right), `duration: 0.8`, `delay`, `ease: 'power3.out'`
- Import gsap/ScrollTrigger from `@/lib/gsap-init`
- Render a styled wrapper div with `ref={containerRef}` and pass through className

- `'use client'` directive
- Same props as PageHero (`imageSrc`, `title`, `subtitle?`, `imagePosition?`)

- Gold accent bar animates width from 0 to 3rem
