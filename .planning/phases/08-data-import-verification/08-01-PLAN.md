# Phase 8 Plan 01: Excel Data & Schema Extension

## Objective
Define the Excel template structure, extend the database schema for real fund data (holdings, quota classes), and validate the Excel files provided by the user.

## Tasks
1. Extend Supabase schema: add `fund_holdings` table, `quota_class` and `residual_commitment` to fund_positions
2. Update TypeScript types in `src/lib/supabase/types.ts`
3. Validate Excel file structure (sheets, columns, data types)
4. Document expected Excel format for future imports

## Success Criteria
- Schema supports all data fields from Excel reports
- TypeScript types match database schema
- Excel files validated and ready for import
