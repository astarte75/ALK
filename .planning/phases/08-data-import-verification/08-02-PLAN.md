# Phase 8 Plan 02: Import Script

## Objective
Build a reusable import script that reads Excel reports, extracts fund data, investors, positions, capital calls, NAV history, and portfolio holdings, and inserts them into Supabase.

## Tasks
1. Build Excel parser for each sheet (Dashboard, Investitori, Cash Flows & IRR, Portafoglio & Fair Value)
2. Implement investor deduplication across funds
3. Calculate pro-rata NAV for investors without explicit CTV NAV
4. Map call descriptions to call_type enum
5. Batch insert with error handling
6. Add --dry-run mode for preview
7. Clear test/seed data before importing real data

## Success Criteria
- Script imports 2 funds with all related data
- Idempotent: re-runnable without duplicates (upsert on slug)
- Dry-run mode prints summary without writing to DB
