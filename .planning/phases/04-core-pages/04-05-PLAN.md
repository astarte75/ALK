1. Create `src/app/api/contact/route.ts`:
   - Export `async function POST(request: NextRequest)`
   - Parse request body as JSON with type:
     ```typescript
     interface ContactPayload {
       firstName: string
       lastName: string
       email: string
       phone: string
       requestType: string
       message: string
       recaptchaToken?: string
     }
     ```
   - Validate required fields — return `NextResponse.json({ error: 'Missing required fields' }, { status: 400 })` if any missing
   - Validate email format with basic regex: `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
   - reCAPTCHA verification (only when configured):
     ```typescript
     if (process.env.RECAPTCHA_SECRET_KEY && body.recaptchaToken) {
       const res = await fetch(`https://www.google.com/recaptcha/api/siteverify?secret=${process.env.RECAPTCHA_SECRET_KEY}&response=${body.recaptchaToken}`, { method: 'POST' })
       const data = await res.json()
       if (!data.success || data.score < 0.5) {
         return NextResponse.json({ error: 'reCAPTCHA verification failed' }, { status: 403 })
       }
     }
     ```
   - SMTP transport (only when configured):
     ```typescript
     if (!process.env.SMTP_HOST) {
       console.log('SMTP not configured. Contact form data:', { firstName, lastName, email, phone, requestType, message: message.substring(0, 100) })
       return NextResponse.json({ success: true, note: 'SMTP not configured — logged to console' })
     }
     ```
   - Create nodemailer transport:
     ```typescript
     const transporter = nodemailer.createTransport({
       host: process.env.SMTP_HOST,
       port: Number(process.env.SMTP_PORT) || 587,
       secure: process.env.SMTP_SECURE === 'true',
       auth: { user: process.env.SMTP_USER, pass: process.env.SMTP_PASS },
     })
     ```
   - Send email:
     - `from`: `process.env.CONTACT_EMAIL_FROM || 'noreply@alkemiacapital.com'`
     - `to`: `process.env.CONTACT_EMAIL_TO || 'segreteria@alkemiacapital.com'`
     - `replyTo`: `email` (the submitter's email)
     - `subject`: `Contatto da ${firstName} ${lastName} — ${requestType}`
     - `html`: Formatted HTML email with all form fields in a clean layout
     - `text`: Plain text fallback with all form fields
   - Wrap in try/catch — return 500 with generic error on transport failure
   - Return `NextResponse.json({ success: true })` on success
   - Do NOT use `export const dynamic = 'force-dynamic'` — route handlers are dynamic by default when using POST
