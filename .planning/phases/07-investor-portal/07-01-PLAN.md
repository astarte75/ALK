- Export async `updateSession(request: NextRequest)` function
- Create a Supabase server client with request/response cookie bridging pattern from RESEARCH.md

- Return the response with updated cookies
- Check if the request path is a protected portal route (contains `/dashboard`, `/fondi/`, `/documenti`)
- If protected route and no user, redirect to login page (locale-aware: `/investitori` for IT, `/en/investors` for EN)

- Keep the existing matcher config
